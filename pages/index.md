## 游玩建议

### 游玩前需要了解的东西

1. 线下实体机基本游玩逻辑（如有什么模式、卡的作用等）
2. 会电脑使用基础
3. 了解连接服务器游玩与离线游玩的区别
4. 了解联网与联机的区别

### 游戏建议最低配置

```
CPU: Intel Core i5-3550S 3.00 GHz  
运存: 8 GB DDR3 2400 MHz  
显卡: NVIDIA GeForce GTX 650 Ti  
```
